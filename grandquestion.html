C0de Killer

Imaginez que vous développiez une application composée de deux parties : un site web et une base de données. Écrivez un fichier docker-compose.yml simple permettant de démarrer ces deux composants. Assurez-vous de spécifier le port sur lequel le site web sera accessible et le nom de l'image Docker pour chaque service.

Les regles du jeu :

Vous disposez de 3 minutes et 30 secondes pour répondre à l'ensemble des questions. Si vous n'y parvenez pas dans le délai imparti, les questions seront automatiquement validées. Donc, votre but est de réussir dans le temps, à condition que la connaissance soit avec vous.

Toute tentative de triche sera sanctionnée, vous avez été prévenu.
Évitez tout raccourci suspect tel que copier-coller ou autre.
